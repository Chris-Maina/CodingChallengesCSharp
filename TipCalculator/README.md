## Tip Calculator
### Challenge
Write a C# function that calculates and returns three different tip percentages for a meal.

**Input**
- Integer value (cost of meal)
**Output**
 - All three percentages as a single value

### Criteria
Tip at 10%
Tip at 17.5%
Tip at 25%
Each rounded off to 2 decimal places.
Example
  CalculateTip(133);
    - 10% -> $13.30
    - 17.5% -> $23.28
    - 25% -> $33.25
